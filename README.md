🍞 Bakery Website

Frontend HTML CSS JavaScript

A responsive business landing page for a local bakery, built to showcase products, location, and brand story.

🔗 Live Demo: https://asmaafathy542.github.io/Bakery-Website/

✨ Features Overview

🏠 Home

Hero/landing section introducing the bakery
Product showcase with item cards

📍 Location

Dedicated location section with map/GPS integration
Easy way for customers to find the store

👋 About

"Who Are You" section telling the bakery's story

⭐ Client Testimonials

Client reviews section to build trust with visitors

🔗 Social & Contact

Social/contact icons (Email, Google, X) for quick reach-out
🛠️ Tech Stack
HTML5 — structure
CSS3 — responsive styling
JavaScript — interactivity
